       - Hiển thị hình ảnh trên trang:
        Sử dụng thẻ img 
            * thuộc tính: 
            + src: url (tương đối, tuyệt đối)
                -> url tuyệt đối: 
                    [giao thức] -> http, https, ... 
                    [domain]    -> devmaster.edu.vn 
                    [path]      -> vị trí file ảnh 
                ví dụ: 
                    https://devmaster.edu.vn/images/logo.png 
                -> url tương đối: trong cùng một website 
                    tính từ vị trí trang hiện tại: 
                        +> vào một thư mục: folder/tên file 
                            VD: images/logo.png 
                        +> ra ngoài một thư mục: ../ 
            + alt: không nên để trống / null
